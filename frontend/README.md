# AQI Intervention Platform — Frontend

This is the React + Vite frontend dashboard for the AQI Intervention Platform.

## Quick Start (Vite Dev Server)

To run the frontend dashboard:

```powershell
# Navigate to the frontend directory
Set-Location -LiteralPath "c:\Users\VINIL NAIK\OneDrive\Desktop\[PUB] India_runs_data_and_ai_challenge\AQI Intervention\frontend"

# Install package dependencies
npm install

# Start Vite dev server
npm run dev
```

*The dashboard will be available at http://localhost:5173/.*

> [!NOTE]
> Make sure the FastAPI backend is running on `http://localhost:8000` for the dashboard to load live AQI data and map elements.
